PHP Forms



HTML Form Element

Interactive websites require input from users. One of the most common ways to get input is with forms. There are two important attributes of <form> tag that are required to add interactivity:

  • Action - This attribute is used to specify where to send the form data when the form is submitted. This attribute can take either an absolute URL or a relative URL.
  • Method - This attribute is used to specify how to submit the form information. It can have only two values; either post or get. The default mathod is get.

Example



POST Method

When the method attribute of <form> is set as post, the $_POST superglobal variable is used. $_POST is an array of variables passed to the current script.

In the POST method, there is no restriction on the data size to be sent and it can be used to send ASCII as well as binary data. The data sent by this method goes through HTTP header and all name-value pairs are embedded within the body of the body of the HTTP request and, hence, the data is invisible and more secure.


Example



GET Method

When the method attribute of <form> is set as get, the $_GET superglobal variable is used. $_GET is same as $_POST.

In GET method, the data size is restricted to 2048 characters only and this method cannot be used to send binary data like images and word documents to the server. This method should not be used for password and other sensitive information as the data is visible in the URL as nam-value pairs while being sent to the server.

Text Files


Opening and Closing Files

The PHP fopen() function is used to open a text file and the fclose() function is used to close an open file.

The fopen() function takes two parameters:

  • The name of the file to be opened.
  • The mode in which the file should be opened.

This function returns a file pointer if successful and if it is unable to open the file, it reuires a 0 (false).

The various file modes are:

Mode Description
r This is the read-only mode and the file pointer is at the start of the file.
r+ This is the read/write mode and the file pointer is at the start of the file.
w This is the write-only mode and the file pointer is at the beginning of the file. It erases the contents of the file or creates a new file if it doesn't exist.
w+ This is the read/write mode and the file pointer is at the beginning of the file. Its functioning is same as that of w mode.
a This is the append mode and the file pointer is at the end of the file. Any existing data is preserved and it the file doesn't exist, it attempts to create it.
a+ This is the read/append mode and the file pointer is at the end of the file. Its functioning is same as that of a mode.
x This is the write-only mode. This mode returns false if the file already exists. If the file doesn't exist, it attempts to create it.
x+ This is the read/write mode. This mode too returns false if the file already exists. If the file doesn't exist, it creates to create it.

Reading from a Text File

Once a file is opened using fopen() function, it can be read with three functions:

  • fread()
  • fgets()
  • fgetc()

The function fread() is used to read speicfied number of bytes and takes two parameters - name of the file and the number of bytes to be read.

The function fgets() is used to read a single line from a file.

The functino fgetc() is used to read a single character from a file.
