Database Linking



MySQL Database

PHP supports many databases likr MS Access, Oracle, DB2. However, the most popular database system used with PHP is MySQL.

MySQL is an open-source, multiuser and multithreaded database management system owned by Oracle. This database is available on most OS platforms.

PHP5 scripts can work with a MySQL database using the MySQLi extension or the PDO (PHP Data Objects).

While PDO is capable of working with 12 different database systems, MySQLi works only with MySQL databases. Earlier versions of PHP used the MySQL extension with was deprecated in 2012.


Opening and Closing Database Connection

PHP provides mysqli_connect() function to open a database connection. This function takes four parameters and returns a MySQL link indentifier on success or FALSE on failure. The four parameters of this function are:

  • servername - An optional parameter that contains the server or host name running the database server. The default value is localhost:3306.
  • username - An optional parameter used to specify the username accessing the database. The default value is the name of the user that owns the server process.
  • password - An optional parameter used to specify the password of the user accessing the database. The default value is an empty password.
  • database_name - An optional parameter used to specify the default to be used when performing queries.

The function mysqli_close() is used to close the database connection when finished retrieving or updating data. By default, the connection is closed automatically when the script ends.

Creating a Database

A database consists of one or more related tables. To create and delete a database, one requires admin privilege.

PHP uses mysqli_query function to execute SQL statements. This function takes two parameters and returns TRUE on success and FALSE on failure. Its parameters are:

  • query - It is a required paramter that contains the SQL query.
  • connection - It is an optional parameter used to specify the connection reference. If it is not specified, the last opened connection of mysqli_connect() is used.

To create a database, the sql argument is CREATE DATABASE database_name; statement.

Inserting Records in a Table

Inserting records means adding or appending a new record at the end of the database. The "insert into" statement is used to add new records to a MySQL table. It has the syntax: INSERT INTO table_name (column1, column2, ...) VALUES (value1, value2, ...);.

There are certain rules for inserting records in a table:

  • Column count and value count should be same
  • If a column has a default value, that column can be omitted
  • Except numberic values and null, other values need to be enclosed in single quotes (' ')

Example



Retrieving Data from a Database

Data is retrieved from a MySQL database using the "select" statement. Its syntax is: SELECT column_name(s) FROM table_name WHERE condition;.

To fetch the entire table, the command "select * from table_name;" is used.

While retrieving multiple columns, column names are seperated by commas.


Example



Updating Data in a Database

Updating records refers to changing data in the existing records. The record(s) to be updated need to be identified beofre the updation takes place.

Data can be updated in MySQL tables by executing the "update" statement. The syntax of this statement is: UPDATE table_name SET column_name=new_value WHERE column_name=old_value;.


Example



Deleting Data from a Database

Deleting records involves removing a record from the database table. Like in update, the record to be deleted reuires to be identified first.

The "delete" statement is used to delete records. The conditional clause "where" is used to locate the record which is to be deleted. If the conditional clause is not used, all the records in the table willl be deleted.

The syntax for the delete command is: DELETE FROM table_name WHERE column_name=value;.
