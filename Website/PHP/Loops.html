Conditional Statements



If... Else Statement

The if...else statement is used to evaluate whether a condition is true or false and, depending on the result, to specify one or more statements to run.

Usually, the condition is an expression that uses a comparison operaotr to compare one value or variable with another.

This conditional statement can be used in three different ways as shown in the example.

Switch Statement

This provides an alternative to the if...elseif statement, providing additional control and readability. This is well suited for situations where there is a large number of possible conditions for the value being checked.

Iterative Statements


For Loop

The for loop is used to perform a loop a specific number of times. It uses a counter variable, which is incremented or decremented with each repition of loop.

The for loop is a deterministic loop because it is predictable, i.e. the number of iterations of the loop are known in advance.

The for loop takes three expressions inside its parentheses, split by semi-colons (;) - initialisation, condition, increment.


Example



Foreach Loop

The foreach loop is similar to for loop but instead of repeating a loop a certain number of times, it repeats the loop for each member of the array.


Example



While Loop

The while loop repeats a block of statements until a specified condition is true.

The while loop is an entry-controlled loop, i.e. the condition is checked at the beginning of the loop. If the condition is false, the loop is not executed even once.


Example



Do...While Loop

Do...While loop is similar to while loop, except that the condition that the condition is tested at the end of each iteration.

This loop is an exit-controlled loop, i.e. the condition is checked at the end of the loop. Even if the condition is false, the loop is executed once.

Arrays


Definition and syntax

An array is a type of variable that groups a series of values an places them in a single variable.

Arrays are useful when storing sets of similar data as arrays make it easier to manipulate the data together. Arrays can store numbers, strings, and any object.

Arrays are defined using the array() function.

There are two types of arrays in PHP:

  • Indexed Array - An array with a numeric index.
  • Associative Array - An array where each ID key is associated with a value.

Indexed Array

The index of these arrays is represented by numbers. By default, indexing starts from zero.

There are two ways to create an indexed array:

  • Assign the index automatically
  • Assign the index manually

Example



Associative Array

These arrays use named keys that are associated with each value in the array.

There are two ways to create an associative array:

  • Specifying all key-value pairs in one go
  • Specifying each key-value pair seperately one-by-one
