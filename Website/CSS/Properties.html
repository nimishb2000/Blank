CSS Properties



Properties

A style sheet is developed by specifying the style definitons, which specifies the formatting characteristics or properties of an element.

The style properties can be:

  • Font Properties
  • Text Properties
  • Background Properties
  • Margin Properties
  • Padding Properties
  • Border Properties
