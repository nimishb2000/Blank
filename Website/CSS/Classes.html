Classes and IDs



Classes

Classes specify additional style definition associated with specific occurences of HTML elements. Classes can be used to create variations for HTML tags.

A class is simply a user-defined label, added as an extension to any HTML tag which a certain style will be applied.

There are two types of classes:

  • Specific class selectors
  • Generic class selectors

IDs

IDs work in the same way as classes,except that they can only be used once.

ID selectors have become the standard in CSS positioning.

Since positioned elements are usually unique on each page, they are distinguished form each other using ID selectors.


Specific Selectors

To start a class/ID, the HTML tag is declared for which the class/ID is required.

Then, either a dot(.) for a class or a hash(#) for an ID is added after the tag without any spaces.

Lastly, the name of the class or id is entered.

Generic Selectors

Generic selectors enable user to use a class or id for various different tags. For example, a class "abc" can be used for both <p> tag as well as <h1> tag.

Using generic selectors is better than using specific selectors as the same styles can be required for different tags. So, intead of defining the styles repeatedly for allt ags, one can create a general class and use it.
