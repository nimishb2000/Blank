HTML Frames



The Frame Tag

HTML frames are portion of webpages inside webpages for them to work like one.

The <body> tag is never used with the <frameset> tag.

There are two types of frames:

  • Frames
  • iFrames

Frame Attributes

The main tag to implement frames is the <frameset> tag.

Frameset has various attributes like rows, cols and border.

The <frameset> tag is followed by the <frame> tag whose attributes are name, target, border and scrolling.

iFrames

An iFrame is an inline frame and is defined using <iframe> tag.

It is used to embed another document within the current HTML document.

Text (if written) between the <iframe> and </iframe> tags is visible only if the user browser does not support iframe.
