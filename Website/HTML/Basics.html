HTML Basic Examples



HTML Documents

All HTML documents must start with a document type declaration: <!DOCTYPE html>.
The HTML document itself begins with <html> and ends with </html>.
The visible part of the HTML document is between <body> and </body>.


Example



HTML Headings

HTML headings are defined with the <Hn> tag where n varies from 1 to 6.
<H1> defines the largest heading while <h6> defines the smallest heading.


Example



HTML Paragraphs

HTML paragraphs are defined with the <p> tag

HTML Links

HTML inks are defined by the <a> tag
The link's destination is defined in the href attribute
Attributes are used to provide additional information about HTML documents


Example



HTML Images

HTML images are defined with the <img> tag
The source file, alternative text, width and height are provided by src, alt, width and height attributes respectively
