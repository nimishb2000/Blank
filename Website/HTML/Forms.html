HTML Forms



HTML Form Element

The HTML <form> element defines a form that is used to collect user input.

The form element provides three tags to create interface elements:

  • <INPUT>
  • <TEXTAREA>
  • <SELECT>

Example



<Input> Tag

The input tag is used to tell the browser where user input will occur.

It specifies most of the elements of a form.

The input tag has three main attributes - type, name and value.

The type attribute defines the type of input that is required from the user and creates the following interface elements:

  • Text Field
  • Password Field
  • Radio Button
  • Check Box
  • Submit Button
  • Reset Button
  • Image Button
  • Simple Button
  • File Field
  • Hidden Field

The name attribute is used to provide each element a specific name which can be used to identify the field by scripts.

The value attribute is used to declare the default value of input field and the text to be displayed on a button.


Example



<Textarea> Tag

Textarea fields are expanded text fields which can span multiple lines and is used for accepting multi-line input from the user.

Unlike most form fields, textareas are defined using <textarea> tag instead of <input> tag.

The <Textarea> tag is a container tag and the text written between <Textarea> and </Textarea> tags is presented in the textarea box.

Scrollbars in a textarea are automatically when visible when input exceeds the visual number of columns or rows specified.

The <Textarea> tag has four attributes:

  • Name - It specifies the name of the textarea
  • Cols - It is used to specify how many characters wide the textarea will be.
  • Rows - It specifies how many visible lines are there in the textarea input.
  • Wrap - It specifies what to do when lines are longer than the width of the textarea.


Example



<Select> Tag

The <Select> tag is used to construct drop-down menus and scrolling-list boxes.

Drop-down menus can act as both radio button and check box, depending on the attribute values.

The advantage of drop-down menu over a raio button or a check box is that it occupies less space as compared to the other two.

The <option> tag is used to specify each item to be displayed in the menu.

The value of the option selected is sent to the server when the form is submitted.

The select tag has three attributes:

  • Name - It specifies the name name of the drop-down menu.
  • Size - It specifies the number of options visible in a scrolling-list box. If this attribute is not used, a drop-down menu is created.
  • Multiple - It is used to specify that multiple options can be selected using Ctrl Key and is only aapplicable in scolling-lists.

The <option> tag has two attributes:

  • Value - It specifies the text to be sent to server if the option is selected.
  • Selected - It specifies default selection to be displayed on page load.
