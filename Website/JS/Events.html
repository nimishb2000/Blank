Javascript Event Handling



Events and Event Handlers

Events are the mechanism by which browsers respond to user actions.

The special functions that are linked to specific events are called event handlers and they execute automatically when events occur. These are made up of the word on and the event they handle.

Events defined by javascript are:

HTML Element Javascript Event Description
Window, Frameset, Frame blur A window loses the current input focus
error An error occurs when window is loaded
focus A window recieves current input focus
load The loading of a window is complete
unload The user closes a window
move A window is moved
resize A window is resized
Document, Body click User clicks in the body of a document
dblClick User double-clicks in the body of a document
keyDown User presses a key
keyUp User releases a key
keyPress User presses and releases a key
mouseDown The mouse button is pressed
mouseUp The mouse button is released
Link Click The mouse is clicked on a link
dblClick User double-clicks on a link
mouseDown The mouse button is pressed
mouseUp The mouse button is released
mouseOut The mouse is moved from inside a link to outside of that link
mouseOver The mouse is moved over a link
keyDown User presses a key when link has focus
keyUp User releases a key when link has focus
keyPress User presses and releases a key
Image abort An image loading is aborted
error An error occurs during loading of an image
load An image is loaded and displayed
click The mouse is clicked on an image
dblClick The mouse is double-clicked on an image
mouseDown The mouse button is pressed
mouseUp The mouse button is released
mouseOver The mouse is moved over an image
Form submit A form is submitted by the user
reset A form is reset by the user

Interacting with HTML Button Element

Javascript can interact with a button element of a form using its properties and events.

The properties of a button element are:

Name Description
form Gets the form object containing the button
name Gets or sets the name of the button
type Gets or sets the type of the element (returns "button")
value Gets or sets the value of the button

The events of a button element are:

Name Description
onclick Triggers when user clicks the button
onfocus Triggers when the button gets the focus
onblur Triggers when the button loses the focus

These properties and events apply to the Submit and Reset buttons too.

Interacting with HTML Text Element

A text box is a rectangular object that recieves or displays text.

The properties of a text element are:

Name Description
default value Gets or sets the default value of the text box
form Gets the form object containing the text box
name Gets or sets the name of the text box
type Gets or sets the type of the element
value Gets or sets the value of the text box

The events of a text element are:

Name Description
onblur Triggers when the text box loses the focus
onchange Triggers when the content of the text box is changed
onfocus Triggers when the text box gets the focus
onselect Triggers when the contents of the text box are selected

Interacting with HTML Radio Button Element

The radio element presents a circular button the user can click. Radio controls are used to allow the user to select only one choice from a list of possible choices. To do this, all the buttons are put in the same group by giving each button the same name.

The properties of a radio element are:

Name Description
checked Gets or sets whether the radio button is selected
defaultchecked Gets or sets whether the radio button is checked by default
form Gets the form object containing the radio button
name Gets or sets the name of the radio button
type Gets or sets the type of the element (returns "radio")
value Gets or sets the value of the radio element

The events of a radio element are:

Name Description
onblur Triggers when a radio button loses the focus
onfocus Triggers when a radio button gets the focus
onclick Triggers when a radio button is clicked

name Gets or sets the name of the checkbox button

value Gets or sets the value of the checkbox element

Name Description

onfocus Triggers when a checkbox button gets the focus

Interacting with HTML Select Element

The select control presents a list from which the user can select one of several items.

The properties of the select element are:

Name Description
form Gets the form object containing the list
length Gets or sets the length of the list
name Gets or sets the name of the list
options An array that identifies options supported by the list
selectedIndex Gets or sets the index of the item selected from the list and the indexing starts from 0
type Gets or sets the type of the element

The events of the select element are:

Name Description
onblur Triggers when the list loses the focus
onfocus Triggers when the list gets the focus
onchange Triggers when an item of the list is selected
