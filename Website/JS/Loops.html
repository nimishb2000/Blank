Conditional Statements



If...Else Statement

The if...else statement is used to add decision making capabilities to the script.

This statement is used to evaluate whether a condition is true or false and, depending on the result, to specify one or more statements to execute.

Usually, the condition is an expression that uses a comparison operaotr to compare one value or variable with another.

This conditional statement can be used in three different ways as shown in the example.

Switch Statement

This provides an alternative to the if...else statement, providing additional control and readability. This is well suited for situations where there is a large number of possible conditions for the value being checked.

Iterative Statements


For Loop

The for loop is used to perform a loop a specific number of times. It uses a counter variable, which is incremented or decremented with each repition of loop.

This loop is suitable in situations where it is known how many times a statement needs to be executed.

The for loop takes three expressions inside its parentheses, split by semi-colons (;) - initialisation, condition, increment.


Example



For...In Loop

The for...in loop works primarily with arrays or objects.

This loop is similar to for loop but instead of repeating a loop a certain number of times, it repeats the loop for each element of a specified collection of items.


Example



While Loop

The while structure loops as long as the condition being checked is true. This loop is useful where the number of times the loop is to be executed is not pre-known.

The while loop is an entry-controlled loop, i.e. the condition is checked at the beginning of the loop. If the condition is false before the loop starts, the loop will never execute.


Example



Do...While Loop

The do...while loop structure repeats a block of statements until a specified condition is met.

This loop is an exit-controlled loop, i.e. the condition is checked at the end of the loop. Even if the condition is false, the loop is executed once.
