Object-Based Programming



Objects

An object is a programming abstractoin that groups data with the code hat operates on it.

Traditionally, codes and data are kept apart but in javascript, code(methods) and data(properties) are merged into a single indivisble unit called object.

Javascript supports two types of objects:

  • Pre-defined built-in objects
  • User-defined objects

Creating an Object

To create an object, the keyword "new" is used and the syntax is: var var_name = NEW object_name([optional_parameters]).

The keyword new tells the javascript that a new object is to be created.


Using Object's Properties

Any object has its unique characteristics. unique characteristics are called ad properties in the object-oriented teminology.

An object's property stores data related to the object and can be referred to using the syntax: object_name.property_name.

The dot(.) is used to seperate the object from its property.

Unlike variables, some properties are read-only, i.e. it is possible to get information from them but it is not possible to change the information inside them.

An object can also be a property of another object, thus creating an object hierarchy. This uses the same syntax: object1.object2property1.property2


Calling an Object's Methods

Methods are functions associated with objects. They enable an object to perform different actions, mostly on its own properties.

They have exclusive access to the state of the object and are, hence, different from normal functions.

A method is called using syntax: object_name.method_name([arguments]).

Since a method is a function, a pair of parentheses always follows the name of the method. This rule applies to all methods, including those which do not have any arguments.


The Array Object

An array is a type of variable that groups a series of values and places them in a single varibale. These are simple built-in objects in javascript and created using the syntax: var array_name = new Array(array_length).

Array length is an optional paramter used to specify the initial length of the array.

The Array object has the following properties:

Property Description
constructor Used to reference the constructor function for the object
length Returns the number of elements in the array

The Array object has the following methods:

Method Description
concat(array2) It concatenates two arrays and returns the array thus formed. The new array returned has elements of first array followed by those of second array.
join(delimiter) Joins all the elements of an array into a single string seperated by the specified delimiter.
reverse() Reverses the order of elements in an array.
sort() Sorts the elements of the array in ascending order.
toString() Converts the Array object into a string with elements seperated by commas.

The Date Object

The date object handles everything to do with date and time in javascript. A Date object can be created in four different ways using the constructor:

  • new Date()
  • new Date(milliseconds)
  • new Date(date_string)
  • new Date(year, month, day, hours, minutes, seconds, milliseconds)

The Date object has the following properties:

Property Description
constructor Used to reference the constructor function for the object

The Date object has the following methods:

Method Description
getDate() Retrieves the date in the month from the Date object.
getDay() Retrieves the day of the week from the Date object as integer, with Sunday as 0 and Monday as 1.
getFullYear() Retrieves the year as a four digit number from the Date object.
getHours() Retrieves the hour of the day form the Date object.
getMinutes() Retrieves the number of minutes from the Date object.
getMonth() Retrieves the month from the Date object as an integer, with January as 0 and February as 1.
getSeconds() Retrieves the number of seconds from the Date object.
getTime() Retrieves the number of milliseconds since January 1 1970 00:00:00 from the Date object.
getYear() Retrieves the year from the Date object.
setDate(Date) Sets the date in the month to the date provided for the Date object.
setFullYear(4_digit_year) Sets the full year to the provided 4 digit year for the Date object.
setHours(hour) Sets the hour of the day to the provided hour for the Date object.
setMintes(Minutes) Sets the number of minutes to the provided number for the Date object.
setMonth(Month) Sets the month to the provided month for the Date object. The month of the year is passed as an integer, where 0 is January and 1 is February.
setSeconds(Sec) Sets the number of seconds to the provided number for the Date object.
setTime(milliseconds) Sets the time for the Date object according to the number of milliseconds since January 1 1970 00:00:00.
setYear(Year) Sets the year to the provided year for the Date object.
toString Converts the Date object into a string.
