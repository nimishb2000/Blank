Javascript Functions



Advantages of Function

A function is a named block of statements that are referenced and executed as a unit to accomplish a specific task or purpose in a program.

The advantages of using function in a program are:

  • It makes the program more structured and breaks down complex programs into small manageable modules.
  • It saves typing effort. If the same set of statements needs to be executed multiple times at various locations, we can simply call the function wherever required.
  • It saves testing and debugging time since it is easier to fetect and remove errors from small modules than from a large complex program.
  • It is easier to make changes as changing code in a function requires change in only one place instead of many cahnges scattered throughout the program.

Defining and Calling a Function

To create a function, the function keyword is used, followed by the name of the function. The block of statements of a function are surrounded in curly braces.

Parameters of the function (if any) are given next to the name of the function inside parentheses. A parameter is an item or data that the function needs to be given to do its job. Even if there are no parameters, the parentheses are mandatory.

Functions are only executed if specifically asked to, which is termed as calling or invoking the function. A function can be called by writing its name at the point where it has to be called.

Built-in Functions

Javascript offers a few functions that are pre-defined or built-in into the javascript interpreter, which can be used directly in any string.

These functions are highly reliable and can tremendously reduce the amount of coding required for a script.

Some of the most common built-in functions are:

Function Return Type Description
parseInt(string) Integer number Converts a string of numbers into an integer value. If a non-integer numeric string is given as argument, the nearest integer is returned.
parseFloat(string) Float number Converts a string of numbers into a floating value.
eval() Number/String It accepts a string representing a number or a mathematical expression and returns the value of the expression after evaluation. If the value supplied is not a number, it returns a string as a concatenated value.
isNaN Boolean It evaluates an arguments to determine if it is not a number (or NaN). It returns true if the argument is not a number and false otherwise.
